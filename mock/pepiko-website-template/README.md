# Pepiko Website HTML/CSS Template

Static HTML/CSS template based on the supplied Pepiko homepage and resources wireframes.

## Included pages

- `index.html`
- `resources.html`
- `case-studies.html`
- `blog.html`
- `guides.html`
- `benchmarks.html`
- `policy-templates.html`

## Folders

- `css/styles.css`
- `assets/img/`
- `assets/reference/` contains the provided reference wireframe images when available.

## How to use

Open `index.html` in a browser, or serve the folder locally:

```bash
python3 -m http.server 8080
```

Then visit:

```txt
http://localhost:8080
```

## Notes

This is a static starter template. Replace placeholder links, logos, benchmark numbers, and example company names before production use.
